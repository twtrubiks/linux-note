## cmatrix

[Youtube Tutorial - 超酷又超沒用的 cmatrix](https://youtu.be/Db0Lz909LNE)

還記得有一部電影叫做駭客任務 (The Matrix) 嗎 :question:

這個軟體就是以他為命名的,

這個軟體沒什麼用, 就裝酷而已 :smile:

安装 cmatrix

```cmd
sudo apt install cmatrix -y
```

使用方法超簡單

```cmd
cmatrix -a
```

![alt tag](https://i.imgur.com/89KIGdE.png)

查看更多的參數

```cmd
cmatrix --help
```
