# Grub Customizer tutorial

## Installation

Add the official PPA

```cmd
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
```

Update

```cmd
sudo apt-get update
```

Install grub-customizer

```cmd
sudo apt-get install grub-customizer
```
