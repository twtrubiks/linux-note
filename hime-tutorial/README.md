## 教學

[hime](https://github.com/hime-ime/hime) 官方 repo 位置,

安裝指令 Debian/Ubuntu

```cmd
sudo apt-get update
sudo apt install hime
```

如果參考我的設定, 切換輸入法快捷鍵為 `Ctrl+Space`,

然後切換到中文輸入法後, 快捷鍵 `Shift` 為可切換英文,

`Shift+Space` 可切換全形或半形.
