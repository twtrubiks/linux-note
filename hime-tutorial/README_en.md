## Tutorial

Official repo location for [hime](https://github.com/hime-ime/hime).

Installation command for Debian/Ubuntu:

```cmd
sudo apt-get update
sudo apt install hime
```

If you see the following icon after installation:



settings:



Select hime:



Recommended settings:





If you follow my settings, the shortcut to switch input methods is `Ctrl+Space`.
Then, after switching to the Chinese input method, the shortcut `Shift` can be used to switch to English.
`Shift+Space` can switch between full-width and half-width.
