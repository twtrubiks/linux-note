Linux 上好用的剪貼簿:thumbsup:

CopyQ 官方文件 [copyq](https://copyq.readthedocs.io/en/latest/installation.html)

ubuntu 18.04 下的安裝方法,

```cmd
sudo apt install software-properties-common
sudo add-apt-repository ppa:hluk/copyq
sudo apt update
sudo apt install copyq
```
